🔒 Security First

This Is Not Spyware.

We understand the concern. An app that can execute commands on your computer sounds dangerous. Here's why Open File is the opposite of malware:

✓
100% Open Source

Every single line of code is publicly available on GitHub under the MIT License. Anyone can audit it.

✓
Zero Telemetry & Zero Data Collection

Open File sends absolutely no data anywhere. No analytics, no tracking, no phone-home. Your API key goes directly from your machine to the AI provider you choose — we never see it.

✓
Built in Rust (Memory Safe)

The entire backend is written in Rust, a language designed to eliminate buffer overflows, use-after-free bugs, and other exploit vectors that plague C/C++ software.

✓
Runs 100% Locally

Open File is a native, standalone macOS application. It does not require a server, cloud account, or internet connection to function (only the AI provider needs internet).

✓
No Hidden Processes

Unlike spyware tools like Predator or Pegasus, Open File does not install kernel extensions, background daemons, or system-level hooks. When you close it, it's closed.

How It Actually Works.

No magic. No tricks. Just a clean abstraction layer between you and your operating system's terminal.

01

You Speak

Type a natural language request into the Command Bridge. Something like: "Organize my Desktop by file type into folders."

→
02

AI Translates

Your chosen AI provider (running on their servers, via your API key) converts your request into a precise macOS shell command.

→
03

Rust Executes

The Tauri Rust backend securely runs the command on your machine using std::process::Command. The output is captured and returned.

→
04

AI Reasons & Repeats

If the task isn't complete, the AI reads the output, decides what to do next, and generates more commands. This loop continues until the objective is fulfilled.
